# Расписание пар

Мой первый React-проект — мобильное приложение для просмотра расписания пар.

## Тема приложения

Расписание пар студента.

## Что есть в проекте

- Заголовок с названием приложения
- Описание
- Три карточки с парами (время, аудитория, предмет, преподаватель)
- Кнопка с интерактивностью (счётчик нажатий + alert)
- Стили для мобильного экрана
- Отображение текущей даты

## Как запустить

```bash
npm install
npm run dev
